# ItunesLyrics
A windows web server to show lyrics for now playing song on itunes.

Based on iTunes-bridge:https://github.com/AngryKiller/iTunes-bridge

# Try to use

- npm install
- open your itunes
- node .\example.js
- open localhost:8080 in your browser
